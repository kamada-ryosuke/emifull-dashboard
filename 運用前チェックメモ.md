# 障がい事業部ダッシュボード 運用前チェックメモ

## ログイン権限

- 管理者: `kamada.rusk@emifull-group.or.jp`
- 一般ユーザー: 現場スタッフ向け。閲覧中心。
- パスワードはハッシュ化してDBに保存する。
- ログイン成功、最終操作、ログアウトはログイン履歴に記録する。

## 管理者が利用できる画面

- ログイン
- 売上一覧 / 入金管理
- 損益ダッシュボード
- 財務 / 経理
- 給与台帳
- 職員台帳
- 車両管理
- 施設マスタ / 設定

## 一般ユーザーが利用できる画面

- ログイン
- 損益ダッシュボード
- 車両管理

## 一般ユーザーの制限

- 売上一覧 / 入金管理、財務 / 経理、給与台帳、職員台帳、施設マスタ / 設定はURL直打ちでも利用不可。
- 損益ダッシュボードでは、現場スタッフに必要なタブを表示する。
  - サマリ
  - 構成比
  - 比較
  - 業績会議
  - 報告書提出
  - 販管費推移
- 業績会議タブは一般ユーザーにも表示するが、Excel出力などの管理者向け出力は管理者のみ。
- 報告書提出タブは一般ユーザーにも表示する。一般ユーザーは自分が提出した報告書を修正できる。
- 車両管理では、一般ユーザーは一覧・アラート・廃車一覧の閲覧のみ。
- 車両の編集、新規登録、車検更新、廃車登録、廃車取消は管理者のみ。

## 運用時の注意

- 終了時は左サイドバー下部の「ログアウト」ボタンを押す。
- Streamlit Cloudがスリープしている場合は、青い復旧ボタンで起動してからログインする。
- データ取込、削除、ユーザー管理、設定変更は管理者が実施する。

## 公開前の簡易確認項目

- 管理者でログインし、全メニューが表示されること。
- 一般ユーザーでログインし、サイドバーが「ログイン」「損益ダッシュボード」「車両管理」のみになること。
- 一般ユーザーで売上一覧、財務、給与、職員、施設マスタのURLを直接開いても利用できないこと。
- 一般ユーザーの損益ダッシュボードで、業績会議・報告書提出タブが表示されること。
- 一般ユーザーの業績会議タブで、数値と報告書プレビューを閲覧できること。
- 一般ユーザーの報告書提出タブで、自分の報告書を提出・修正できること。
- 一般ユーザーの車両管理で、編集・新規登録・廃車取消ボタンが表示されないこと。
- ログアウト後、ブラウザの戻る操作で業務画面を操作できないこと。
- CSVやExcelを開いたときに文字化けしないこと。
